ghcmod¶

ghcmod¶
check¶

check a file for GHC warnings and errors

Example:

{
    "params": {
        "file": {
            "file": "a file name"
        }
    },
    "cmd": "ghcmod:check"
}
lint¶

Check files using hlint

Example:

{
    "params": {
        "file": {
            "file": "a file name"
        }
    },
    "cmd": "ghcmod:lint"
}
info¶

Look up an identifier in the context of FILE (like ghci’s :info)

Example:

{
    "params": {
        "expr": {
            "text": "The EXPR to provide info on"
        },
        "file": {
            "file": "a file name"
        }
    },
    "cmd": "ghcmod:info"
}
type¶

Get the type of the expression under (LINE,COL)

Example:

{
    "params": {
        "start_pos": {
            "line": 42,
            "col": 23
        },
        "file": {
            "file": "a file name"
        }
    },
    "cmd": "ghcmod:type"
}
